# Arcane Go SDK

[![Go Reference](https://pkg.go.dev/badge/github.com/anujc4/arcane-go.svg)](https://pkg.go.dev/github.com/anujc4/arcane-go)

Official Go SDK for the [Arcane API](https://arcane.privaterelay.xyz) - Modern Docker Management, Designed for Everyone.

This SDK is automatically generated from the OpenAPI specification and provides a type-safe, idiomatic Go interface for interacting with the Arcane API.
